# Boomi Salesforce コネクタ接続手順書

## 概要

Boomi Integration から Salesforce（Developer Edition）へ SOAP API を使用して接続するための設定手順を記載します。

> **注意**: Salesforce は SOAP API login() を将来的に廃止予定（Summer '27 予定）です。新規の本番環境では OAuth 2.0 への移行を検討してください。本手順は開発・検証環境向けです。

---

## 前提条件

- Salesforce Developer Edition のアカウントを持っていること
- 対象ユーザーのプロファイルが「システム管理者」など、API アクセス権限を持つこと
- Boomi Atomosphere へのアクセス権があること

---

## 手順1: Salesforce で SOAP API login() を有効化する

新しく作成された Salesforce 組織では、SOAP API login() がデフォルトで無効になっています。以下の手順で有効化します。

1. Salesforce にログインする
2. 右上の歯車アイコンから **「設定（Setup）」** を開く
3. 左側のクイック検索ボックスに **「ユーザインターフェース」** と入力する
4. **「トランスレーションワークベンチ」** 配下にある **「ユーザインターフェース」** をクリックする（※ 同名のグループ項目ではなく、サブ項目の方を選択すること）
5. 「設定」セクション内にある **「SOAP API login()を有効化」** にチェックを入れる
6. **「保存」** をクリックする

---

## 手順2: Salesforce のセキュリティトークンを取得する

1. Salesforce にログインした状態で、右上のユーザーアイコンをクリックする
2. **「設定（Settings）」** を選択する（※ 管理者用の「Setup」ではなく、個人設定の方）
3. 左側メニューの検索ボックスに **「トークン」** と入力する
4. **「私のセキュリティトークンのリセット」** をクリックする
5. **「セキュリティトークンのリセット」** ボタンをクリックする
6. 登録メールアドレスに届いたセキュリティトークンを控えておく

---

## 手順3: Boomi でコネクションを設定する

Boomi Atmosphere にログインし、Salesforce コネクションを以下のとおり設定します。

| 項目 | 設定値 |
|------|--------|
| **URL** | `https://login.salesforce.com/services/Soap/u/61.0` |
| **ユーザー名** | Salesforce のログインユーザー名（メールアドレス形式） |
| **パスワード** | `Salesforceのパスワード` + `セキュリティトークン`（スペースなしで続けて入力） |
| **バージョン** | 15.0 - 最新（ドロップダウン固定。URL でバージョンを指定しているため問題なし） |

### パスワード入力例

- パスワード: `MyPassword123`
- セキュリティトークン: `xYz9AbCdEfGh`
- 入力する値: `MyPassword123xYz9AbCdEfGh`

---

## 手順4: 接続テスト

1. コネクション設定を保存する
2. Salesforce インポートウィザードなどで接続を確認する
3. エラーなく Salesforce のオブジェクト一覧が表示されれば接続成功

---

## トラブルシューティング

| エラーメッセージ | 原因と対処 |
|----------------|----------|
| `INVALID_LOGIN` | ユーザー名、パスワード、またはセキュリティトークンが正しくない。パスワード＋トークンが隙間なく繋がっているか確認する。 |
| `UNSUPPORTED_API_VERSION` | API バージョンが古すぎる。URL のバージョン番号を 31.0〜64.0 の範囲で指定する。 |
| `INVALID_OPERATION: SOAP API login() is disabled` | SOAP API login() が無効。手順1 に従い有効化する。 |
| `LOGIN_MUST_USE_SECURITY_TOKEN` | セキュリティトークンが未付加。パスワードの後にトークンを続けて入力する。 |

---

## 補足: API バージョンに関する注意

- SOAP API login() は **バージョン 31.0〜64.0** でのみ利用可能です
- バージョン 65.0 以降では SOAP API login() は廃止されており、使用できません
- Salesforce は **Summer '27（2027年中頃）** に全バージョンで SOAP API login() を完全廃止する予定です
- 長期的には **OAuth 2.0** への移行を推奨します
